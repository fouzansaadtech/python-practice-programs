# 🐍 Python Practice Programs-Week 2

A collection of Python practice programs built while learning programming fundamentals during my Data Analytics journey. This repository contains beginner-friendly exercises focused on problem-solving, loops, conditional statements, strings, lists, and tuples.

---

## 📚 Topics Covered

- Python Basics
- Conditional Statements
- Loops (`for` & `while`)
- String Operations
- List Operations
- Tuple Operations
- Problem Solving

---

## 📂 Programs Included

### 📋 List Programs
1. Largest Number
2. Even & Odd Counter
3. Search Item in List
4. Remove Duplicates
5. Separate Positive & Negative Numbers
6. Reverse List (Without `reverse()`)

### 🔤 String Programs
7. Reverse String (Without Slicing)
8. Uppercase & Lowercase Counter
9. Vowel & Consonant Counter
10. Print Character with Index

### 📦 Tuple Programs
11. Sum & Average of Tuple
12. Count Occurrence of an Element
